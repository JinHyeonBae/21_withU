- 구성도

## 배경
독거노인들의 가족과 단절되어 살아가기 때문에, 이에 대한 고독감에서 야기되는 정서적인 문제점들이 많다. 이러한 현상에서 착안하여,
독거노인들과의 교류가 가능한 시스템을 만들어, 독거노인들의 정서 건강을 증진시키고자 하였다.

## 기술 스택
- App
  - React native, Node.js
- Natural language processing
  - Python, seq2seq
- aduino, raspberry pi
  - python
